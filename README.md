# test_frontend_v2

本專案為 Nuxt 3 + TypeScript 前端範例，實作用戶 CRUD、Pinia 狀態管理、i18n 語系切換、RWD、元件封裝與 SSR。

## 主要功能

- 用戶資料 CRUD（增刪改查）
- SSR 取得用戶資料
- Pinia 狀態管理
- axios API 請求
- i18n 語系切換（繁體中文/英文）
- 元件封裝（EBtn、ETextField）
- Dialog 確認操作
- RWD 響應式設計
- vue-tsc 型別檢查
- Vercel 自動化部署
